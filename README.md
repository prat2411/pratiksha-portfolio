# Pratiksha Wankhade Portfolio

A modern single-page portfolio website built with React and Vite.

## Live Website

Once GitHub Pages deployment finishes, the site will be available at:

https://prat2411.github.io/pratiksha-portfolio/

## Tech Stack

- React
- Vite
- CSS (custom responsive design)

## Sections Included

- Hero
- About
- Projects
- Skills
- Education
- Contact

## Local Development

1. Install dependencies:

   npm install

2. Start development server:

   npm run dev

3. Open the local URL shown by Vite (typically http://localhost:5173/).

## Production Build

npm run build

## Deployment

This repository includes a GitHub Actions workflow that automatically deploys the site to GitHub Pages whenever changes are pushed to main.
